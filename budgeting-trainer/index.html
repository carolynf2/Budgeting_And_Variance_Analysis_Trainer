Budget Variance Types

Favorable Variance

Revenue: Actual > Budget (Good performance)

Expenses: Actual < Budget (Cost savings)

Example: Budgeted marketing $5,000, actual $4,200 = $800 Favorable

Unfavorable Variance

Revenue: Actual < Budget (Missed targets)

Expenses: Actual > Budget (Cost overruns)

Example: Budgeted sales $50,000, actual $45,000 = $5,000 Unfavorable

Common Variance Causes

Revenue Variances

  • Market demand changes
  • Price adjustments
  • Competitor actions
  • Seasonal fluctuations
  • Product quality issues

Expense Variances

  • Inflation/price increases
  • Volume changes
  • Efficiency improvements
  • Unexpected repairs
  • Staffing changes
